Multi-Segmenter
다음은 모든 클래스 멤버들의 목록입니다. 각 멤버들은 해당 멤버가 속해 있는 클래스에 대한 링크를 가지고 있습니다. :

- a -

  • adj_triangles : FaceGraph
  • ambient : Material

- d -

  • diffuse : Material
  • dissolve : Material
  • do_segmentation() : OriginalSegmenter , ParallelSegmenter , Segmenter , SerialSegmenter

- e -

  • emission : Material

- f -

  • FaceGraph() : FaceGraph

- g -

  • get_segments() : FaceGraph , ParallelFaceGraph , SerialFaceGraph

- i -

  • illumination_model : Material
  • index : TriangleMesh

- m -

  • material : TriangleMesh
  • material_texture_id : TriangleMesh
  • mesh : Segmenter
  • meshes : Model

- n -

  • name : Material , TriangleMesh
  • normal : TriangleMesh

- o -

  • operator()() : Vec3Hash
  • OriginalSegmenter() : OriginalSegmenter

- p -

  • ParallelFaceGraph() : ParallelFaceGraph
  • ParallelSegmenter() : ParallelSegmenter
  • pixel : Texture

- r -

  • read_obj() : Model
  • resolution : Texture

- s -

  • Segmenter() : Segmenter
  • SerialFaceGraph() : SerialFaceGraph
  • SerialSegmenter() : SerialSegmenter
  • shininess : Material
  • specular : Material

- t -

  • texcoord : TriangleMesh
  • textures : Model
  • tolerance : Segmenter
  • traverse_dfs() : FaceGraph , ParallelFaceGraph , SerialFaceGraph
  • triangles : FaceGraph

- v -

  • vertex : Triangle , TriangleMesh

- ~ -

  • ~Model() : Model
  • ~Texture() : Texture
  • ~TriangleMesh() : TriangleMesh
