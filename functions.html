Multi-Segmenter
다음은 모든 클래스 멤버들의 목록입니다. 각 멤버들은 해당 멤버가 속해 있는 클래스에 대한 링크를 가지고 있습니다. :

- a -

  • adj_triangles : FaceGraph::FaceGraph
  • ambient : Material

- c -

  • check_connected() : FaceGraph::FaceGraph

- d -

  • diffuse : Material
  • dissolve : Material
  • do_segmentation() : OriginalSegmenter , ParallelSegmenter , Segmenter , SerialSegmenter

- e -

  • emission : Material

- f -

  • FaceGraph() : FaceGraph::FaceGraph

- i -

  • illumination_model : Material
  • index : TriangleMesh

- l -

  • link : FaceGraph::GraphNode

- m -

  • material : TriangleMesh
  • material_texture_id : TriangleMesh
  • mesh : Segmenter
  • meshes : Model

- n -

  • name : Material , TriangleMesh
  • normal : TriangleMesh

- o -

  • operator()() : FaceGraph::Vec3Hash
  • optical_density : Material
  • OriginalSegmenter() : OriginalSegmenter

- p -

  • ParallelSegmenter() : ParallelSegmenter
  • pixel : Texture

- r -

  • read_obj() : Model
  • resolution : Texture

- s -

  • Segmenter() : Segmenter
  • SerialSegmenter() : SerialSegmenter
  • shininess : Material
  • specular : Material

- t -

  • texcoord : TriangleMesh
  • textures : Model
  • tolerance : Segmenter
  • traverse_dfs() : FaceGraph::FaceGraph
  • triangles : FaceGraph::FaceGraph , FaceGraph::GraphNode

- v -

  • vertex : FaceGraph::Triangle , TriangleMesh

- ~ -

  • ~GraphNode() : FaceGraph::GraphNode
  • ~Model() : Model

  • ~TriangleMesh() : TriangleMesh
